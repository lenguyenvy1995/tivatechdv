# Settings Menu Editor

`Settings Menu Editor` is a Botble plugin to manage Settings page section/order from a single screen.

---

## Features

- Drag and drop both `Settings` and `System` section titles.
- Drag and drop menu cards inside a section.
- Move cards between sections in the same block.
- Add, edit, and delete custom headings.
- Choose whether a new custom heading belongs to `Settings` or `System`.
- Hide/show section headings.
- Hide/show individual menu cards.
- Keep empty custom headings visible.
- Reset saved layout to default.
- Save results so order is applied on `/admin/settings`.

---

## Where To Access

- Admin URL: `/admin/settings/settings-menu-editor`
- Permission required: `settings-menu-editor.index`
- Panel item title: `Settings Menu Editor`

---

## Usage

1. Open `Settings Menu Editor` from admin settings.
2. Drag section headers to reorder section groups in each block.
3. Drag cards to reorder inside a section.
4. Add a custom heading and choose target block (`Settings` / `System`).
5. Click `Save Order`.
6. Go to `/admin/settings` and `/admin/system` and verify applied order.
